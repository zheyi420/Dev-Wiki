# JTS

JTS 是加拿大的 Vivid Solutions 公司做的一套开放源码的 Java API。它提供了一套空间数据操作的核心算法。为在兼容 OGC 标准的空间对象模型中进行基础的几何操作提供 2D 空间谓词 API。
